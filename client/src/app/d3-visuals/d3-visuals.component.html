D3 Visualizations View various types of D3 Visualizations created based on board games and average ratings below by selecting a type to expand.
Choropleth Map Average Rating of Board Games Across the World

The Choropleth map shows the average rating per country using a color quantile scale based on the board game selected in the dropdown. Hover the mouse over the country to see additional info. If the country does not have any data, then it is gray.

 Force-Directed Graph Layout Demonstrates the similarity between board games based on categories and game mechanics.

Edges (similarity): The edge value is the total weighted similarity index. If the board games (nodes) are similar, then the edge is gray, thick, and solid. If they are not similar, then the edge is green, thin, and dashed.

Nodes (board games): Radius and color gradient of each node is based on degree of the node. Darker color and larger radius mean higher degree.

Note: You can pin the nodes by clicking and dragging. The color of the node will change to red when pinned. To unpin, double click the node.
