ANALYSIS

The color compression above is performed using k means clustering. This means that the pixels are grouped into k color (RGB) clusters based on the pixels distance from the cluster center.

You get to select the k value for the number of clusters that will be used. The initial k cluster centers are randomly selected pixels in the image. Then, the iterative loop begins. Each pixel is added to a color cluster based on which cluster center it is closest to. Once all th pixels are grouped, the new cluster centers are cacluated. This will continue until the cluster centers are no longer changing. Essentially, the pixels are not changing clusters anymore.

Finally, all the pixels are assigned the RGB value from its cluster center, the image is reshaped, and you now get an image with k colors.
